## 2. 즉시 보강 필요 (P0) — 광고 시작 전 필수

### 2.1 ❌ sitemap.xml / robots.txt 없음
**왜 중요**: 구글봇이 페이지 발견 못 함 → 자연 검색 유입 0.
**영향**: 광고 종료 후 SEO 자산 못 쌓음.

**해결**:
```typescript
// app/sitemap.ts
export default function sitemap() {
  return [
    { url: 'https://ozlabpay.kr/', priority: 1.0 },
    { url: 'https://ozlabpay.kr/blog', priority: 0.8 },
    // 블로그 글 동적 추가 (Phase 컨텐츠 작업 후)
  ]
}

// app/robots.ts
export default function robots() {
  return {
    rules: { userAgent: '*', allow: '/', disallow: '/admin' },
    sitemap: 'https://ozlabpay.kr/sitemap.xml',
  }
}
```
→ Next.js 14 표준 패턴. 30분 작업.

### 2.2 ❌ OG 이미지 없음
**현재 상태**: layout.tsx 의 OG 메타에 `images` 누락. 카톡·페이스북 공유 시 이미지 안 떠 클릭률 저하.
**해결**:
- Phase 1: 정적 OG 이미지 1장 (1200x630) — `app/opengraph-image.png` 추가만으로 자동 적용
- Phase 2: 동적 OG (페이지마다 다른 이미지) — `app/opengraph-image.tsx` (Next.js 14 ImageResponse)

**즉시 카피 + 이미지 제안**:
- 검은 배경 + 네이버 네온 그린
- 큰 글씨: "POS + 단말기 0원"
- 서브: "오즈랩페이"

### 2.3 ❌ metadataBase 누락
**현재 코드**:
```ts
// layout.tsx — metadataBase 없음
```
**문제**: OG 이미지·canonical URL이 절대 URL 안 됨 (상대 경로만).
**해결**: `metadataBase: new URL('https://ozlabpay.kr')` 추가.

### 2.4 ❌ JSON-LD 구조화 데이터 없음
**왜 중요**: 구글 리치 결과(별점·FAQ 박스·로컬 비즈니스 카드) 노출 못 함.
**필요한 스키마 4종**:
- `Organization` — 회사 정보
- `WebSite` — 사이트 검색 박스
- `LocalBusiness` 또는 `Service` — POS·단말기 서비스
- `FAQPage` — FAQ 섹션 (이미 있음, 마크업만 추가)

→ 광고 직전에 추가하면 1주일 내 구글 리치 결과 노출 시작.

### 2.5 ❌ GA4 미연동 (가장 치명적)
**현재**: gtag 스크립트 없음. `consultations.ga_client_id` 컬럼은 만들어뒀지만 **실제로 채워지지 않음**.
**영향**: 광고 어트리뷰션 분석 불가능. 메타·구글 광고 효율 측정 못 함.

**즉시 작업 (1시간)**:
1. GA4 측정 ID 발급 (`G-XXXXXXXXXX`)
2. layout.tsx 에 gtag 스크립트
3. ApplyForm 제출 시 `client_id` 추출 → consultations 에 저장
4. (Phase B) GA4 → BigQuery 익스포트로 멀티터치 어트리뷰션

### 2.6 ❌ 광고 픽셀 미설치
- **메타 픽셀** (페이스북·인스타 광고)
- **구글 ads 전환 추적**
- **카카오 픽셀**
- **틱톡 픽셀**

각 광고 플랫폼에서 발급 → layout.tsx 에 스크립트 추가. **각 매체마다 광고 시작 전 필수**. 안 하면 광고 최적화 못 함.

## 3. 중요 보강 (P1) — 광고 1주차 안에

### 3.1 🟡 신뢰 신호 부족
**현재**: Testimonials 1개 섹션만. 고객 후기·매장 사진·도입 매장 수 없음.
**보강안**:
- "현재 N개 매장 도입 중" 카운터 (실시간 업데이트 가능)
- 도입 매장 로고 5~10개 (눈에 익은 프랜차이즈 1~2곳)
- 영상 후기 (유튜브 임베드)
- 언론 보도·인증 마크 (있으면)

### 3.2 🟡 광고 랜딩 분리 X
**현재**: 모든 광고가 `ozlabpay.kr/` 한 페이지로 도달.
**문제**: 메타 광고와 구글 광고는 **타겟 의도 다름** — 단일 랜딩이 양쪽 모두에 비효율.
**제안**:
- `/ad/meta` — "쉽게 시작" 강조 (메타 콜드 트래픽)
- `/ad/google` — "단말기 0원" 키워드 매칭 (구글 검색)
- `/ad/place` — "플레이스 광고 무료" (네이버 검색)
- `/ad/danggn` — "동네 자영업자" (당근)

각 페이지는 카피만 다르고 폼은 공통. UTM 자동 첨부.

### 3.3 🟡 페이지 속도 미측정
**필요**: Lighthouse / PageSpeed Insights 점수 확인.
**예상 이슈**:
- Pretendard CDN font 로드 (fcp 영향)
- 이미지 최적화 (Next.js Image 사용? 확인 필요)
- 미사용 CSS

→ 광고 시작 전 1회 측정 후 점수 90+ 목표.

### 3.4 🟡 폼 UX 개선 여지
**현재 ApplyForm**: 7개 필드 (이름·연락처·매장·업종·지역·메시지·동의)
- ✅ 필수만 ★ 표시 (이름·연락처·동의)
- ✅ honeypot 봇 차단
- ⚠️ **개선**:
  - 모바일에서 한 화면에 다 안 들어가 — **2단계 분할** 검토 (1단계: 연락처만 / 2단계: 상세)
  - 자동 포커스: "성함" 필드에 첫 진입 시 X
  - 전화번호 자동 포맷 (010-xxxx-xxxx)
  - 제출 후 카톡 채널 추가 유도 (즉시 전환률 ↑)

### 3.5 🟡 가격 섹션 모호
**현재 Pricing**: 3개 카드 가격 명시.
**확인 필요**:
- "무료 단말기" 조건 (계약 기간·최소 결제액 등) — FAQ로 명시
- 환불·해지 정책
- 추가 비용 (수수료·통신비)

→ 광고 직후 "숨은 비용" 의심 댓글 다수 예상. 명시로 차단.

## 5. 광고 시작 전 체크리스트 (P0 통합)

```
[ ] sitemap.xml + robots.txt 추가 (30분)
[ ] OG 이미지 1장 + metadataBase 설정 (1시간)
[ ] JSON-LD 구조화 (Organization + LocalBusiness + FAQPage) (1시간)
[ ] GA4 측정 ID 발급 + layout 통합 + ApplyForm 클라이언트 ID 캡쳐 (2시간)
[ ] 메타 픽셀 + 구글 ads 전환 추적 픽셀 설치 (각 매체 광고 계정 만든 후 30분씩)
[ ] (선택) 광고 매체별 랜딩 페이지 분리 (P1, 광고 시작 후라도 OK)
```

→ 이거 다 하고 광고 시작하면 **첫 클릭부터 어트리뷰션 100% 보임**.
→ 안 하고 시작하면 첫 1~2주 데이터는 "유입은 봤는데 어디서 왔는지 모름" 상태.
